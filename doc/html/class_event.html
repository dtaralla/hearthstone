Member Enumeration Documentation

enum Event::Type

The Type enum contains all event types fired by the game engine.

Enumerator
NO_EVENT 

Represents no event. This will never be fired.

Extras:
  • None
AFTER_CARD_PLAYED 

Fired after any card is played.

Extras:
  • card: [Card] The card played
Note
Minions can be played from a player's hand but also directly summoned on the battlefield by some actions. These summon effects do not fire this event.
AT_TURN_END 

Fired at all turn ends.

This event is fired before the control is switched to the other player, and after Abilities::FROZEN conditions are removed.

Extras:
  • player: [Player] The player whose turn is ended
BEFORE_CARD_PLAYED 

Fired before any card is played.

Extras:
  • card: [Card] The card played
Note
Minions can be played from a player's hand but also directly summoned on the battlefield by some actions. These summon effects do not fire this event.
CHARACTER_ATTACKS 

Fired at each attack action resolution.

This event is fired before the actual attack occurs (before the attacker connects to its target).

Extras:
  • attackingCharacter: [Character] The attacker
  • attackedCharacter: [Character] The initial target
CHARACTER_DAMAGED 

Fired whenever a character is damaged.

This event is fired after the damages are done but before any death resolution.

Extras:
  • damager: [Card] The source of the damages if any, NULL otherwise.
  • damagedCharacter: [Character] The damaged character
CHARACTER_HEALED 

Fired whenever a character is healed.

This event is fired after the healing is done.

Extras:
  • healer: [Card] The source of the healing if any, NULL otherwise.
  • healedCharacter: [Character] The damaged character
MINION_DIES 

Fired whenever a minion dies.

This event is fired at the end of the resolution phase, when deaths are resolved one by one. Deaths are resolved in the order of play of the dying minions. This event is fired after the resolution of the dying minion's deathrattles, if any.

Extras:
  • dyingMinion: [Minion] The dying minion
