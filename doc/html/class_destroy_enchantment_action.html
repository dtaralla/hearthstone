Asks this Action to resolve itself.

Executes the action by modifying the game state accordingly.

Calling this method can result in a chained series of events, triggering other Actions.

Warning
One should always call resolvable() first, and ensure it returns true before calling this method. Failing to do so might end up in undefined behaviour.
