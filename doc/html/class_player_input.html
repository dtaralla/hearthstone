virtual void PlayerInput::onEventTriggered ( IORequest *  ir)
pure virtualslot

onEventTriggered Called whenever something is triggered (not each time an event occurs; only when there is something to trigger). The response set by this slot will be ignored by the signalling context; it should not set any else a memory leak could occur. NB: We need this slot to be synchronous, because "event" and "action" extras might be destroyed before these slots are executed!

Parameters
irA request IORequest::TRIGGER, with extras:
  • Event* "event": The event that occured.
  • Action* "action": The action which is triggered
