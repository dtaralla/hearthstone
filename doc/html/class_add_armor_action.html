Detailed Description

The AddArmorAction class represents an Action whose effect is to increase (or decrease) the armor of heroes.

Author
David Taralla
Date
2015

User-selection of the hero is not possible. The hero whose armor has to be modified is determined at runtime, thanks to an Owner qualifier like

  • Owners::ALLY : The target will be the allied hero upon calling resolve()
  • Owners::ENEMY : The target will be the enemy hero upon calling resolve()
  • Owners::ANY_OWNER : Both heroes will get the effect

The quantity of armor point to give or remove is a second parameter for this action.
